Timer
    для учёта в логгере информации о функции в начале функции создаётся экземпляр класса Timer, в конструктор передаётся id функции
    таймер замеряет время своего создания и время разрушения
    при разрушении экземпляра класса в деструкторе вызывается функция добавления информации о работе функции в логгер

Logger - singleton class
    получить экземпляр логгера можно через GetInstance
    выдаёт функции id через TakeId и имя функции
    собирает информации о работе функций в вектор log_info через AddInfo при разрушении экземпляра класса Timer
    при разрушении в конце программы печатает в файл лог в формате: имя функции, время работы, количество запусков

TimeInfo - структура, описывающая функцию
    содержит имя функции, общее время работы, количество вызовов
